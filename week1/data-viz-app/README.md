# ChartSlice

A small interactive bar chart that lets you **slice the data by category**. Built for the Spring Into AI Build Sprint — Week 1 theme: data visualization and interactions.

## What it is

Single-page app: bar chart of sample monthly data with a dropdown to filter by category (All / A / B). Demonstrates **data visualization** (chart) and **interactions** (filter). React + Vite + Recharts.

## How to run it

```bash
npm install
npm run dev
```

Open the URL shown (e.g. http://localhost:5173).
